Note for v1: To upload the firmware correctly, you have to add the Sparkfun SAMD21 Mini board definition to your Arduino installation. Follow the instructions in this link:

https://learn.sparkfun.com/tutorials/samd21-minidev-breakout-hookup-guide/setting-up-arduino

Note for v2-3: In the Arduino IDE 2.X, you'll need to install support for Teensy boards from the boards manager on the left sidebar.
